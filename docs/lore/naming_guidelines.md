## 序文 — なぜ命名ガイドラインが必要か

GrassRoots世界観の大原則は **「サッカー界はオリジナル、世界史は現実」** である。実在のサッカー選手・監督は存在せず、全員創作キャラクターでなければならない。

しかしEra 1〜10の命名作業中、以下の踏み越えが発見された:

- **実在選手の名前を直接使用**: エウゼビオ、ロナウド等
- **姓名のモジリ(捩り)**: エトゥー(エトオ)、中田英樹(中田英寿)等
- **特定選手の有名エピソードを描写**: 頭突き、神の手 等
- **実在の非選手との完全同名**: ヴィリー・ブラント(元西独首相)等

本ガイドラインは、これらを**今後一切発生させない**ための命名基準を明文化する。

## 第1章 — 絶対NGルール(Hard Rules)

以下はどのような文脈でも許容されない。

### 1-1. 実在サッカー選手の姓または名と一致してはならない

**NG例:**
- 「ロナウド」「エウゼビオ」「ジダン」「ペレ」「マラドーナ」「メッシ」「クライフ」等、名・愛称レベルでも一致する場合
- 「中田」「本田」「中村」「香川」等、有名日本人選手の姓を日本人キャラに使用する場合
- 「エトオ」「エトゥー」等、カタカナ表記を少し変えるだけの回避

**判定基準:**
- プレイヤーが一目見て「これは○○のことだ」と特定できる名前は、たとえ綴りが違ってもNG
- 有名度の閾値: **サッカーファンなら誰でも知る選手**(W杯優勝メンバー、バロンドール受賞者、各国代表レジェンド等)

### 1-2. 実在選手の有名エピソードを描写してはならない

**NG例:**
- 「頭突きで終わった」(ジダンのマテラッツィ事件特定)
- 「神の手と5人抜きの両方」(マラドーナ1986特定)
- 「17歳で世界を知った怪物」「黒豹」(特定選手のニックネーム使用)

**OK例:**
- 「無冠で終わった仏系マグレブの王」(世代と出自の抽象化)
- 「1986年W杯優勝の神秘的な10番」(時代と役割のみ)

**判定基準:**
- 当該Eraの象徴的な選手像は描写して良いが、**一人に絞り込める記号**(特定の試合・事件・あだ名)を使わない
- 「その時代にそういう選手が何人かいた」レベルまで抽象化する

### 1-3. 現役・最近引退したトップ選手を連想させてはならない

特に注意が必要なのは:
- **2010年代以降のスター選手**(プレイヤーの記憶が新鮮)
- **各国代表の絶対的エース**(サッカーファンなら全員知る)
- **バロンドール級の選手**

### 1-4. 実在の著名非サッカー人との完全同名を避ける

**NG例:**
- ヴィリー・ブラント(元西独首相と完全同名)
- ウィンストン・チャーチル(元英首相)
- ジョン・レノン(音楽家)

**判定基準:**
- **Wikipedia日本語版で単独ページが立つレベルの著名人**との同名は避ける
- ただし、歴史的に特定不能な一般名(トーマス・スミス等)は問題なし

## 第2章 — 推奨ルール(Soft Rules)

### 2-1. 名前は「その国・その時代の実在者名簿から無作為に選んだ」ような質感を目指す

良い名前の条件:
- 国籍と時代に整合する(Era 1英国人にカタカナっぽい名前を付けない等)
- 一般的な姓名の組み合わせ(極端に珍しい綴りや、明らかに作り物っぽい響きを避ける)
- 発音しやすい(プレイヤーが読めない/覚えられない名前は避ける)

### 2-2. 家系キャラは血統の一貫性を保つ

同じ家系の中で、世代を超えて姓を一貫させる。名は各世代で独自に選ぶ。

**例: ブラックウッド家**
- 初代エドマンド → 2代ジェームズ → 3代ウィリアム → 4代アーネスト → 5代ロバート → 6代カーティス → 7代ライアン → 8代ナイジェル → 9代デヴィッド → 10代ジョージ

名前のバリエーションは英国の歴代の一般的な男性名から取る。

### 2-3. 傍流・女性キャラは姓を継承するが、名は出身地域の影響を受けてよい

**例: シルヴァ家**
- 本流(ブラジル): ジョアン、パウロ、アントニオ、アデマール、ペドロ...
- アルゼンチン分家: ディエゴ、ルイス、アレハンドロ(アルゼンチン風)
- 女性分家(ブラジル): マリア

### 2-4. Era時代ごとの命名傾向

| Era | 特徴 |
|:-:|---|
| 1-3 | 古風な名前(クラシカルな英語圏名、ヴィクトリア朝風) |
| 4-5 | 戦間期の名前(各国で時代色あり) |
| 6-8 | 戦後の標準的名前 |
| 9-13 | 現代的名前、移民2世など多様化 |

## 第3章 — 名前の作り方(実務ガイド)

### 3-1. 推奨プロセス

1. **キャラ設計を先に固める**: Era、国籍、家系、役割、物語的位置
2. **候補名をリストアップ**: その国の歴代一般名を3〜5個用意
3. **実在選手チェック**: 候補名の姓・名それぞれをチェック
   - 「名+姓」でGoogle検索して実在選手がヒットしないか
   - 各国代表の歴代メンバー表と照合(可能な範囲で)
4. **実在非選手チェック**: 元首・作家・俳優との同名回避
5. **描写文チェック**: 一行要約に特定選手のエピソードが含まれていないか

### 3-2. 国別の命名リソース参考

| 蘇 | ハミッシュ、アンガス、イアン、ダンカン等 | フィオナ、ローナ、アイリーン等 | マクドナルド、キャンベル、スチュワート等 |

## 第4章 — すでに採用した名前の整合性保持

### 4-1. 採用済みキャラのリスト管理

Era 1〜10で採用した154人の名前は**確定**とする(修正済みの一部を除く)。新キャラ追加時は以下を確認:

1. 採用済みキャラと同じ姓名の組み合わせにしないこと
2. 家系キャラの場合、既存の家系メンバーと名が被らないこと

### 4-2. 保護された正典キャラ

以下は既存正典(World Bible/文化設定書)で確定しており、変更してはならない:

- ハミッシュ・マクファーレン(Era 1)

- ステファン・コヴァチェヴィッチ(Era 4)
- ドロシー・ブラックウッド(Era 4)

## 第6章 — 運用

### 6-1. チェックリスト(新キャラ作成時に必ず確認)

- [ ] 姓名の組み合わせで実在サッカー選手をGoogle検索 → ヒットしない
- [ ] 名単独でスター選手ニックネームと被らない(ロナウド/メッシ/ペレ等)
- [ ] 姓単独で著名選手の姓と被らない(中田/本田/エトオ等)
- [ ] 一行要約に特定選手のエピソードが含まれていない
- [ ] Wikipediaで同名の著名非選手(政治家・著名人)がいないか確認
- [ ] 採用済み154人と重複していない
- [ ] 家系キャラの場合、家系内の既存メンバー名と被らない

### 6-2. 運用拡張時の方針

将来Era 11-13追加、または運用で新キャラを増やす際、**本ガイドラインを毎回参照する**こと。疑わしい名前は採用前に代替案を必ず用意する。
